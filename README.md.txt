# 🎮 Number Guessing Game

This is a simple Python game where the computer generates a random number and the user tries to guess it within limited attempts.

## 🚀 Features

* Random number generation
* User input handling
* Hints (Too High / Too Low)
* Attempt counter
* Maximum 5 attempts

## 🛠️ Technologies Used

* Python

## 📚 What I Learned

* Variables
* Loops (while loop)
* Conditional statements (if-else)
* User input handling
* Basic problem-solving logic

## ▶️ How to Run

1. Open terminal
2. Run:
   python main.py
